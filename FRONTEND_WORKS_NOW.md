## ⚠️ ВОЗМОЖНЫЕ ПРОБЛЕМЫ

### 1. "Не играет"

**Причина:** Сервер не запущен

**Решение:**
```bash
cd /home/c1ten12/music-app/backend
source venv/bin/activate
python -m uvicorn main_lite:app --host 0.0.0.0 --port 8000
```

### 2. "CORS Error"

**Причина:** Прокси не работает

**Решение:** Проверьте что endpoint доступен:
```bash
curl http://192.168.31.97:8000/audio/proxy/dQw4w9WgXcQ
```

### 3. "Треки не находятся"

**Причина:** YouTube API не работает

**Решение:** Проверьте прокси:
```bash
ps aux | grep proxy
```
