# 🎵 Ultimate Music App

**Музыкальная платформа нового поколения с системой обхода цензуры**

---

## 🌟 О приложении

Ultimate Music App — это современная музыкальная платформа, которая даёт вам доступ к оригинальным версиям треков без цензуры.

### 🔥 Главная особенность

**Anti-Censorship Система** — автоматически распознаёт цензурированные версии песен и находит оригинальные (explicit) версии на YouTube, SoundCloud и других площадках, даже если названия отличаются.

---

## ✨ Возможности

### 🎵 Музыка

- **Мульти-платформенный поиск** — YouTube, SoundCloud, VK, Navidrome
- **Умное распознавание** — определяет clean/radio/edit версии
- **Поиск оригиналов** — находит explicit версии с помощью fuzzy matching
- **Персональные рекомендации** — на основе ваших предпочтений
- **Умные миксы** — Daily Mixes, Release Radar, Discover Weekly

### 🤖 AI Сервисы

- **Suno AI** — генерация песен по промпту
- **LALAL.AI** — разделение на стемы (вокал, бас, ударные)
- **ElevenLabs** — синтез речи и вокала
- **Mubert** — фоновая музыка

### 📱 Интерфейс

- **Веб-версия** — работает в любом браузере
- **Telegram Mini App** — встроенное приложение в Telegram
- **Адаптивный дизайн** — удобно на любом устройстве

---

## 🚀 Как это работает

1. **Вы ищете музыку** — через поиск или рекомендации
2. **Система проверяет** — есть ли цензура в треке
3. **Автоматически находит** — оригинальную версию на других площадках
4. **Предлагает замену** — если найдена версия без цензуры

## 📡 API

- **Music search** — поиск по всем источникам
- **Recommendations** — персональные подборки
- **AI generation** — генерация музыки
